<a name="presentation"></a>
## 1. Обзор

**PDF Editor** — бесплатный редактор PDF с открытым исходным кодом, который позволяет:

- Читать и просматривать любые файлы PDF
- Редактировать существующий текст прямо в документе
- Вставлять текст, изображения и аннотации
- Создавать и заполнять PDF-формы
- Применять оптическое распознавание символов (OCR) к отсканированным страницам
- Изменять порядок страниц, объединять и разделять документы
- Собирать новый PDF из изображений (JPG, PNG, TIFF…)
- Добавлять колонтитулы, водяные знаки и штампы
- Редактировать метаданные и сжимать файл
- Защищать документ паролем
- Подписывать цифровой подписью с помощью сертификата `.pfx`

<a name="installation"></a>
## 2. Установка и первый запуск

### Портативное приложение

Приложение не требует установки. Дважды щелкните по `PDFEditor.exe`, чтобы запустить его.

### Установка с помощью установщика

Если у вас есть файл `PDFEditor-Setup.exe`, запустите его и следуйте инструкциям мастера.
На одном из этапов предлагается автоматически установить **Tesseract OCR** (необходим для распознавания текста).

Установщик также может **назначить PDF Editor приложением по умолчанию** для открытия PDF-файлов (опция включена по умолчанию).

### Первый запуск — Tesseract OCR

При первом запуске, если Tesseract не обнаружен на компьютере, появится окно с предложением автоматически скачать и установить его (~50 МБ).

- **Язык OCR**: выберите основной язык ваших документов (система автоматически определяет язык Windows).
- **Английский** всегда включен как резервный язык.
- Вы можете отказаться от установки; функция OCR будет недоступна до тех пор, пока Tesseract не будет установлен вручную.

<a name="interface"></a>
## 3. Общий интерфейс

```
┌─────────────────────────────────────────────────────────────────┐
│  Меню  (Файл · Правка · Вид · Инструменты · Справка)            │
├─────────────────────────────────────────────────────────────────┤
│  Основная панель  (◀ Пред. | № стр. / всего | След. ▶ | Масштаб)│
├─────────────────────────────────────────────────────────────────┤
│  Панель страниц  (Изменить порядок/Объединить · Разделить · Удалить …)│
├─────────────────────────────────────────────────────────────────┤
│  Панель аннотаций  (Выбор · Редактировать текст · Выделить · …) │
├──────────────────────┬──────────────────────────────────────────┤
│                      │                                          │
│  Боковая панель      │         Просмотрщик PDF                  │
│  [Страницы]          │         (текущая страница)               │
│  [Инструменты]       │                                          │
│                      │          — или —                       │
│                      │                                          │
│                      │         Непрерывный просмотр             │
│                      │         (вертикальная прокрутка)        │
│                      │                                          │
├──────────────────────┴──────────────────────────────────────────┤
│  Строка состояния                                               │
└─────────────────────────────────────────────────────────────────┘
```

- **Левая боковая панель**: две вкладки — *Страницы* и *Инструменты*. Скрывается клавишей `F4`.
- **Просмотрщик**: одна страница по умолчанию или режим *непрерывной прокрутки* (`Вид → Непрерывная прокрутка`, `Ctrl+Shift+C`).
- **Панель поиска**: появляется в верхней части области чтения (`Ctrl+F`) и закрывается клавишей `Esc`.
- **Строка состояния**: контекстные сообщения, номер страницы, индикатор несохраненных изменений (`*`).

### Вкладки боковой панели

| Вкладка | Содержимое |
|---------|------------|
| **Страницы** | Миниатюры для навигации — щелкните, чтобы перейти к странице |
| **Инструменты** | Раздел *Инструменты* (в том же порядке, что и меню) · раздел *Аннотации* · раздел *Горячие клавиши* |

### Главная строка меню

| Меню | Основное содержимое |
|------|---------------------|
| **Файл** | Открыть, Сохранить, Печать, Выход |
| **Правка** | Отменить, Повторить, Поиск, **Настройки** |
| **Вид** | Масштаб, Панель, Непрерывная прокрутка, Полноэкранный режим, Тема, Внешний вид |
| **Инструменты** | Действия, сгруппированные по подменю: *Вставка*, *Упорядочить*, *Извлечь*, *Защита*, *OCR* |
| **Справка** | Руководство, Горячие клавиши, Сообщить об ошибке, Проверить обновления, О программе |

> Язык, лицензия, интеграция с Windows и внешний вид теперь централизованно находятся в меню **Правка > Настройки**.

<a name="modifier-texte"></a>
## 8. Редактирование существующего текста

PDF Editor позволяет редактировать текст непосредственно в потоке документа.

### Шаги

1. На панели аннотаций выберите инструмент **Редактировать текст** (`T`).
2. **Дважды щелкните** слово или текстовый блок, который нужно отредактировать.
3. Появится всплывающее окно с текстом и параметрами форматирования:
   - Шрифт, размер, **Полужирный**, *Курсив*, цвет, межбуквенный интервал
   - Цвет фона (по умолчанию прозрачный)
4. Отредактируйте текст, настройте форматирование, затем нажмите **Подтвердить** (`Ctrl+Enter`).

> **Совет**: инструмент сначала пытается выполнить **встроенное** редактирование в потоке PDF. Если это невозможно (неизвестный шрифт, текст в изображении), используется замещающая аннотация.
>
> Если редактирование действительно невозможно применить (нередактируемый шрифт, текст в изображении, пустой блок, сбой внедрения в поток), в нижней части окна появляется **постоянное сообщение** в **строке состояния** (например: *«Невозможно отредактировать: встроенное редактирование не удалось…»*).

### Отмена

`Ctrl+Z` — отменить · `Ctrl+Y` — повторить (см. [§27 Отмена / Повтор](#annuler)).

<a name="inserer-texte"></a>
## 9. Вставка текста

Чтобы добавить новый текстовый блок на пустую область:

1. Выберите инструмент **Редактировать текст** (`T`).
2. **Дважды щелкните** пустую область страницы.
3. Откроется всплывающее окно с пустым редактором.
4. Введите текст, выберите форматирование, затем подтвердите.

Текст вставляется в PDF как постоянная аннотация **FreeText**.

<a name="annotations"></a>
## 10. Аннотации

Панель аннотаций предоставляет несколько инструментов:

| Инструмент | Горячая клавиша | Назначение |
|------------|-----------------|------------|
| Выбор | `S` | Выбор и перемещение существующих аннотаций |
| Редактировать текст | `T` | Редактирование текста документа (см. §8 и §9) |
| Выделение | `H` | Выделение слова или выделенного фрагмента желтым цветом |
| Комментарий | `C` | Добавление заметки (пузырька) на странице |
| Изображение | `I` | Вставка изображения (см. §11) |
| Ластик | `E` | Удаление аннотации щелчком |

Эти же инструменты доступны на вкладке **Инструменты** левой боковой панели, раздел *Аннотации* (по умолчанию свернут — щелкните, чтобы развернуть).

### Толщина линии

В панели *Инструменты → Аннотации* поле **Толщина** задает толщину обводки для аннотаций-рисунков (от 0,5 до 10 пт).

### Изменение размера / перемещение аннотации

В режиме **Выбор** (`S`):
- **Щелкните** аннотацию, чтобы выделить ее (появятся маркеры).
- **Перетащите**, чтобы переместить · **Перетащите маркер**, чтобы изменить размер.
- Клавиша `Delete` удаляет выделенную аннотацию.

Изображение встраивается в PDF постоянно.

<a name="formulaires"></a>
## 12. PDF-формы

### Включение режима дизайна

Нажмите **✏ Режим дизайна** на панели *Страницы и форма*.
В режиме дизайна щелчок с перетаскиванием по странице создает новое поле.

### Доступные типы полей

| Тип | Описание |
|-----|----------|
| Текст | Поле свободного ввода |
| Флажок | Да / Нет |
| Выпадающий список | Выбор из предварительно заданных вариантов |
| Переключатели | Единственный выбор в группе |
| Метка | Статический нередактируемый текст |

### Заполнение формы

В обычном режиме (режим дизайна отключен) щелкните поле, чтобы заполнить его.
Боковая панель отображает все поля со их значениями.

### Перемещение поля

*Инструменты → ↔ Переместить текстовый блок* (`M`), затем перетащите поле.

<a name="ocr"></a>
## 13. Оптическое распознавание символов (OCR)

**Необходимое условие**: установленный Tesseract OCR (см. [§2](#installation)). В установочной версии для Windows Tesseract поставляется в комплекте.

### Запуск OCR

1. *Инструменты → OCR → 🔤 Распознавание символов (OCR)…*
2. Справа откроется панель OCR.
3. Выберите **язык** документа.
4. Нажмите **Запустить OCR**.

### Поддерживаемые языки

| Язык | Код |
|------|-----|
| Français (французский) | `fra` |
| English (английский) | `eng` |
| Deutsch (немецкий) | `deu` |
| Español (испанский) | `spa` |
| Italiano (итальянский) | `ita` |
| Português (португальский) | `por` |
| Русский | `rus` |

> Невидимый текстовый слой использует **полную кодировку Unicode** (UTF-16): кириллица, CJK, арабский и все другие алфавиты корректно индексируются (`Ctrl+F`) и поддерживают копирование — без транслитерации и потери символов.

### Результат

- Распознанный текст отображается поверх страницы цветными блоками.
- Размер и положение каждого блока можно изменить перетаскиванием.
- Нажмите **Встроить в PDF**, чтобы сделать текст постоянным.

> Встроенные OCR-блоки невидимы на экране, но индексируются программами чтения PDF (`Ctrl+F`, копирование…).

### Дополнительные параметры OCR

| Параметр | Описание |
|----------|----------|
| *Инструменты → OCR → Восстановить страницу с собственным текстом* | Заменяет текстовые элементы страницы собственным текстом PDF (лучшее качество редактирования). |
| *Инструменты → OCR → Применить исправление как изображение* | Исправляет OCR-текст путем прямого изменения изображения страницы (экспериментально). |

### Исправление строки OCR двойным щелчком

В отсканированном PDF, который уже содержит OCR-слой (например, после нажатия **Встроить в PDF**), можно исправить строку прямо из просмотрщика:

1. Убедитесь, что активный инструмент — **Редактировать текст** (`T`) или **Выбор** (`S`).
2. **Дважды щелкните** отсканированную строку, которую нужно исправить.
3. На месте строки появится **встроенное** поле редактирования (без всплывающего окна).
4. Исправьте текст.
5. Нажмите **Enter** для подтверждения или **Esc** для отмены.

> Исправление сохраняется как невидимая OCR-аннотация, индексируемая поиском (`Ctrl+F`) и копированием. Эту операцию можно **отменить** с помощью `Ctrl+Z`.

> **Совет**: двойной щелчок — самый быстрый способ исправить опечатку в скане. Если строка не распознается при щелчке, сначала запустите *Инструменты → OCR → Распознавание символов (OCR)…*, а затем нажмите **Встроить в PDF**.

<a name="pages"></a>
## 14. Управление страницами — изменение порядка / объединение / разделение

### Изменение порядка и объединение страниц

*Инструменты → Упорядочить → ⊕ Изменить порядок/Объединить страницы…* (или кнопка **⊕ Изменить порядок/Объединить** на панели инструментов)

Этот универсальный инструмент работает **как с открытым документом, так и без него**:

| Ситуация | Результат |
|----------|-----------|
| PDF открыт | Изменяет порядок страниц текущего документа |
| PDF не открыт | Создает новый PDF с нуля |

#### Интерфейс органайзера

- Страницы отображаются в виде **миниатюр**, которые можно менять местами перетаскиванием.
- Выберите одну или несколько миниатюр, затем используйте кнопки:

| Кнопка | Действие |
|--------|----------|
| ▲ Вверх / ▼ Вниз | Переместить выделенное |
| ↺ −90° / ↻ +90° / ↕ 180° | Повернуть выделенные страницы |
| 🗑 Удалить | Удалить выделенные страницы |
| ➕ Добавить документ… | Вставить страницы из другого документа |

#### Добавление документа

Кнопка **➕ Добавить документ…** принимает:
- **PDF** — добавляются все страницы
- **Изображения**: JPG, JPEG, PNG, BMP, TIFF (включая многостраничный), WebP — каждое изображение становится страницей

> **Совет**: чтобы **объединить** несколько PDF, откройте органайзер без открытого документа, добавьте файлы через «Добавить документ», упорядочьте их, затем нажмите **Применить** — появится диалог «Сохранить как» с запросом имени нового PDF.

> Эту операцию можно **отменить** с помощью `Ctrl+Z`.

### Дублирование текущей страницы

*Инструменты → Упорядочить → Дублировать текущую страницу* или `Ctrl+Shift+P`.

> Эту операцию можно **отменить** с помощью `Ctrl+Z`.

### Удаление текущей страницы

*Инструменты → Упорядочить → 🗑 Удалить текущую страницу* или `Ctrl+Delete`.

> Эту операцию можно **отменить** с помощью `Ctrl+Z`.

### Быстрое поворот текущей страницы

| Действие | Способ |
|----------|--------|
| Повернуть +90° | *Инструменты → Упорядочить → ↻ Повернуть страницу (+90°)* или `R` |
| Повернуть −90° | *Инструменты → Упорядочить → ↺ Повернуть страницу (−90°)* или `Shift+R` |

### Разделить этот PDF

1. *Инструменты → Упорядочить → ✂ Разделить этот PDF…*
2. Введите количество **страниц на файл** (например, `1` — по одному файлу на страницу, `5` — группы по 5 страниц).
3. Предварительный просмотр показывает, сколько файлов будет создано.
4. Выберите папку назначения и подтвердите.

<a name="entetes"></a>
## 15. Колонтитулы

*Инструменты → ☰ Колонтитулы…*

Добавьте автоматический текст вверху и/или внизу каждой страницы.

### Текстовые зоны

Каждая зона (Колонтитул и Подвал) имеет три колонки: **Слева · По центру · Справа**.

### Динамические токены

Вставьте переменные, которые будут автоматически заменены при применении:

| Токен | Вставляемое значение |
|-------|----------------------|
| `{page}` | Номер текущей страницы |
| `{total}` | Общее количество страниц |
| `{date}` | Сегодняшняя дата (дд/мм/гггг) |

Кнопки-ярлыки под каждым полем позволяют вставить эти токены одним щелчком.

### Общие параметры

| Параметр | Описание |
|----------|----------|
| Размер шрифта | От 6 до 36 пт |
| Цвет | Черный, Серый, Красный, Синий |
| Отступ от края | Расстояние в мм от края страницы |
| Не применять на 1-й странице | Полезно для титульных страниц |

### Изменение или удаление

Повторно откройте *Инструменты → ☰ Колонтитулы…*: последние использованные настройки загружаются.
- **Изменить**: измените тексты и снова нажмите **Применить** — старые колонтитулы будут заменены.
- **Удалить**: очистите все поля и нажмите **Применить** — колонтитулы будут стерты.

> Эту операцию можно **отменить** с помощью `Ctrl+Z`.

## 18. Графический штамп — логотип и подпись

*Инструменты → Вставка → 🖼 Графический штамп…*

Применяет изображение (логотип компании, отсканированную подпись, печать…) на одной или нескольких страницах. Добавленные штампы **сохраняются между сеансами** в личной библиотеке.

<a name="windows"></a>
## 19. Интеграция с Windows — контекстное меню

*Правка → Настройки → Лицензия и интеграция → Интеграция Windows (контекстное меню)…*

PDF Editor предлагает два пункта в контекстном меню Проводника Windows, оба **включаются автоматически** при первом запуске. Управление ими осуществляется через *Правка → Настройки → Лицензия и интеграция*.

### Пункты контекстного меню

| Пункт | Действие |
|-------|----------|
| **Открыть с помощью PDF Editor** | Открывает выбранный файл в приложении. |
| **Объединить в PDF Editor** | Выбор нескольких файлов → открывает диалог изменения порядка страниц с предварительно загруженными PDF, готовыми к объединению. |

### Преобразование изображения в PDF

Преобразует файл изображения в PDF одним щелчком правой кнопкой мыши (фоновая обработка, без интерфейса).

**Использование**

1. Щелкните правой кнопкой по файлу изображения в Проводнике.
2. Выберите **Преобразовать в PDF — PDF EDITOR**.
3. PDF создается в **той же папке**, с тем же базовым именем (`.pdf`).
4. В конце появляется подтверждение.

**Форматы**: JPG · JPEG · PNG · BMP · TIFF · TIF · WebP

> Если PDF с таким же именем уже существует, добавляется числовой суффикс (`file_1.pdf`, `file_2.pdf`…).

---

### Объединить файлы в PDF Editor

Открывает диалог **Изменить порядок/Объединить** с несколькими предварительно загруженными файлами. Идеально подходит для быстрой сборки PDF и/или изображений в один документ.

**Использование**

1. В Проводнике Windows **выберите несколько файлов** (Ctrl+щелчок или Shift+щелчок).
2. **Щелкните правой кнопкой** по выделенному.
3. Выберите **Объединить в PDF Editor**.
4. PDF Editor открывается и отображает диалог изменения порядка страниц со всеми файлами, предварительно загруженными в виде миниатюр.
5. При необходимости измените порядок страниц, затем нажмите **Применить** и сохраните.

**Поддерживаемые форматы**: PDF · JPG · JPEG · PNG · BMP · TIFF · TIF · WebP

> Пункт также появляется при щелчке правой кнопкой по одному совместимому файлу.
> Изображения автоматически преобразуются во временный PDF перед отображением в диалоге.

## 21. Сжатие PDF

*Инструменты → ⚡ Сжать PDF*

Уменьшает размер файла за счет оптимизации внутренних потоков PDF (сжатие объектов и повторное сжатие существующих данных).

- Сжатие применяется **немедленно** к открытому документу.
- Сообщение в нижней части экрана указывает достигнутое уменьшение размера (в КБ или МБ).
- Не забудьте сохранить (`Ctrl+S`), чтобы сохранить результат.

> Эффект более заметен на неоптимизированных PDF (экспорт из Word, сканы…). Уже сжатые PDF почти не изменятся.

<a name="signature"></a>
## 23. Цифровая подпись

PDF Editor может подписывать документ с помощью цифрового сертификата `.pfx` / `.p12`.

### Доступ

- Через меню: *Инструменты → Защита → ✍ Подписать документ…*
- Через боковую панель: вкладка **Инструменты** → *✍ Подписать документ…*

### Подписать

1. *Инструменты → Защита → ✍ Подписать документ…*
2. Заполните:
   - **Путь к сертификату**: файл `.pfx` или `.p12`
   - **Пароль** сертификата
   - **Причина** и **Место** (необязательно)
   - **Страница**, на которую поместить видимую подпись
3. Нажмите **OK**.

### Проверка подписей

*Инструменты → Защита → 🔎 Проверить подписи…* (или кнопка *🔎 Проверить подписи…* на панели Инструменты) отображает список подписей и их статус действительности.

### Получение сертификата `.pfx`

*Справка → 🔑 Как получить сертификат .pfx?* объясняет возможности:
- Сертификат от Удостоверяющего Центра (Certum, Sectigo, GlobalSign…)
- Самоподписанный сертификат с помощью OpenSSL (только для внутреннего использования)
